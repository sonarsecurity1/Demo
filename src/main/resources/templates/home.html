À propos de cette application
Concepts démontrés :
  • Modèle (Model) : Classe Compte représentant les données
  • Vue (View) : Templates Thymeleaf pour l'affichage
  • Contrôleur (Controller) : CompteController
  • Service Layer : CompteService pour la logique métier
  • Injection de dépendances : @Autowired
  • Gestion de transactions : @Transactional
  • Extraction des paramètres : @RequestParam
  • Mapping d'URL : @GetMapping, @PostMapping
  • Binding de données : @ModelAttribute
